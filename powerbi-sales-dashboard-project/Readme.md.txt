# Power BI Sales Analytics Dashboard

## Project Overview

This project analyzes retail sales data using Power BI.
The goal is to identify trends in sales performance, customer behavior, and product profitability.

## Dashboards Created

1. Sales Performance Dashboard
2. Customer Analysis Dashboard
3. Product Performance Dashboard

## Key Metrics

* Total Sales
* Total Profit
* Total Orders
* Profit Ratio

## Tools Used

* Power BI
* DAX Measures
* Data Visualization

## Key Insights

* Technology category generates the highest sales.
* Consumer segment contributes the largest revenue share.
* Some product sub-categories produce negative profit.
* Overall profit margin is around 12%.

## Dataset

Superstore sales dataset used for analysis.
